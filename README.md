# Capstone Project (React Native / Expo)

This package contains artifacts for the Capstone submission (28 tasks).

## Quick Start
1. Install Expo CLI if needed:
   npm i -g expo-cli

2. Create an Expo project (if you don't have one yet):
   npx create-expo-app capstone-app
   cd capstone-app

3. Copy the /src files from this package into your project.
4. Install dependencies:
   npm i @react-navigation/native @react-navigation/native-stack
   npx expo install react-native-screens react-native-safe-area-context
   npx expo install @react-native-async-storage/async-storage

5. Start the app:
   npx expo start

Structure:
- /docs: screenshots and user_stories.md
- /src: feature implementations
